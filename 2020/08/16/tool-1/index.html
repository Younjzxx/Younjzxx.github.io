jenkins工具

发表于 2020-08-16

jenkins可以做什么

1.jenkins是一种持续集成/持续部署的工具
2.通常结合SVN或者Git版本管理工具，以及项目构建工具Maven等配合使用，配置钩子程序之后即可以达到发布之后就自动调用Maven打war包，部署发布的版本到tomcat服务器的一系列操作。

jenkins与Git集成

服务器安装编译Git时需要的包

1.安装环境
yum install -y curl-devel expat-devel gettext-devel openssl-devel zlib-devel

yum install -y gcc perl-ExtUtils-MakeMaker

2.删除已有的Git
yum remove git

3.Git官网下载最新的tar包，移动到/usr/src目录下
cd /usr/src
tar -zxvf git-xxxx.jar.gz

4.编译安装
cd git-xxx
make prefix=/usr/local/git all
make prefix=/usr/local/git install
配置环境变量
echo “export PATH=$PATH:/usr/local/git/bin” >> /etc/bashrc
source /etc/bashrc

5.检查git版本
git –version

jenkins配置git客户端的位置

github上面添加webhook(钩子)

打开github–>settings–>webhooks–>add webhook
配置下地址即可完成

下次只要push即可自动发布app。

jenkins全局安全配置处取消 防止跨站点请求伪造的勾选
